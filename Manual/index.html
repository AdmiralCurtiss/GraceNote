Prerequisites

GraceNote runs in Python. If you already have Python 2.7.x and PyQt4 installed, you're good. Otherwise:

  • Download and Install Python. On Windows (maybe on other systems too?), be sure you use a 2.x.x version and the x86 installer -- not the x86-64 one!
  • Download and Install PyQt4. Use the version that corresponds to the version of Python you installed. So if you picked a Windows x86 Python 2.7, you want PyQt-Py2.7-x86-gpl-*.exe.
  • Optionally, you can also download and install PyEnchant. This gives you spellchecking support.

Running GraceNote

Execute GraceNote.pyw with Python. Depending on your installation and OS, it might be enough to double-click the file, or you might need to open up a command line and type "python GraceNote.pyw".

GraceNote looks for project configuration files in the Project subfolder. If only one exists, it is automatically loaded. Otherwise you can choose one.
